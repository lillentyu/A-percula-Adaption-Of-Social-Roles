#Plasticity Associated with Adoption of Social Roles in Clown Anemonefish

This repository contains all the code (Code), data files (Data), figures (Figures), and supplementary material associated with genotyping (Genotyping) for the manuscript for the manuscript Vizer et al., Plasticity Associated with Adoption of Social Roles in Clown Anemonefish, in prep

Raw fastq files can be accessed at: SRA BioProject PRJNA1280023.

The main growth, appetite, and behavior script to follow for the analysis of the data for this project is called "percula_phentypic_analysis_20250428.Rmd", for main gene expression analysis can be found in "percula_phenotypic_plasticity_GE.Rmd" script, and main pathway analysis can be found in pathway_analysis20250229.R script, other scripts referenced in the rmd are available in the Code directory.
